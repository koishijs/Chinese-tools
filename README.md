# koishi-plugin-chinese-tools

[![npm](https://img.shields.io/npm/v/koishi-plugin-chinese-tools?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-chinese-tools)

一个汉语工具合集，提供了：

- 一个指令用于手动处理文本
- 一个服务用于其他插件调用
- 一个中间件用于 PUA 用户

目前具备的功能有：

- Koishi 首字母大写
- 繁体转简体
- 简体转繁体
- 盘古插空格
- 全角英文、数字转成半角字符
- 中文前后的半角标点转成全角标点
- 连续的句号自动转省略号
- 感叹号、问号最多允许连续重复 3 次
- 合理删除重复的标点
- 剔除不可见空白
